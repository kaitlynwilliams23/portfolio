Data Science Fellow - Brown University

Project Overview

During the fall semester in 2023, I got the unique opportunity to become a Data Science Fellow at Brown University, where I was paired with two professors, Dr. Liz Chen and Dr. Neil Sarkar, to help integrate data science into their course Methods in Informatics and Data Science for Health in the Brown Center for Biomedical Informatics. This semester long project allowed me to create a couple of guides for their students to use to understand how to read and access health data and how to perform basic analysis on the data.

Timeline: September 2023 - December 2023

Context

For the course Methods in Informatics and Data Science for Health, there is a broad range of students taking the class. Some students have a lot of programming experience, and others have never coded before. As someone who was new to coding my sophomore year of college, I went through the learning process on using Github and creating simple programs, to doing complex machine learning analysis. I was tasked to make guidebooks on the following topics for people of all coding backgrounds, especially those who have never coded before, using what I learned from my experiences learning to code:

  • Github Basics
  • Health Data and Health Data Standards
  • Accessing Electronic Health Records (EHR)
  • Introduction Machine Learning: SciKitLearn, Matplotlib

This project was structured by creating two-three week long sprints throughout the semester to complete, coupled with weekly check in meetings to give feedback, ask for advice, and share my work.

GitHub Basics

In this guidebook, I walked through different git commands that are useful for students to know, as well as included best Github practices for working as an individual or in a group. Some of the git commands I focused on were:

  • git push
  • git pull
  • git commit -m"message"
  • git add -A

Reflection

I am very greatful for the opportunity to work with and learn from Dr. Liz Chen and Dr. Neil Sarkar.

Throughout this process, I wanted to be able to create content that was interesting to students who had previous coding experience, while teaching students who have no previous coding experience how to use data in their analysis. As somebody who started computer science "later in life", I am thankful that I got the opportunity to pass along tips and tricks that aided me in my learning process.

I learned a lot about electronic health data and its benefits and pitfalls, as well as how to create content for people of all different skill levels. I also learned that I really enjoyed a work environment that has weekly meetings to be able to get immediate feedback on my work, as well as encourage more collaboration between me, Dr. Chen, and Dr. Sarkar.
