Development

Project Overview

I created an interactive interface through React using interactive interface components. I wanted to look at creating a clothing selection similar to online shopping websites that have a favorite items list that users can add items to while they are browsing. Additionally, my interface will be designed to have the ability to filter by type of clothing and color, as well as sort by price or alphabetically.

I replicated my favorite features from some of my most frequented online shopping websites, using React to design a shopping site where the user can add items to their "Favorites" list and filter through different clothing options.

Background and Research

What problem did you solve? Why should we care?

To start, I completed a competitive analysis to determine what I liked and didn't like about the "add to favorites" feature from three of my most frequented online shopping websites: Princess Polly, Pacsun, and Garage.

I analyzed the "favorite" feature on all three websites to create the competitve comparison chart below:

I identified the following features I wanted to include in my design to ease the process of adding an item to favorites:

  • Visible button to click to add to favorites list that changes when the item is in the list
  • Obivious Remove button for the list
  • Organize additions from least recent to most recent

Process

I started with a sketch to plan what I wanted the interface to look like. Based off of the three websites I examined earlier, I wanted to make sure that the functionality of elements on the screen were clear to the user.

Key features that I knew I had to include were:

  • two filtering categories
  • at least one sorting feature
  • an aggregaor
  • a reset button to remove filters and sorts
  • items that have a visible button to add them to the favorites list

Final Product

I used React components to create the filtering and sorting, as well as creating an item list that holds the items added to the favorites list. I then added the functionality in the app by creating functions to handle the changes made by pressing buttons.

In order to make the functionality of each button on the screen clear to each user, I made sure to add labels that could guide the user through the interface.
